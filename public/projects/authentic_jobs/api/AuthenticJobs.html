- (nil) search(options = {})

List current positions

Parameters:

  • (Hash) options (defaults to: {}) —

    a customizable set of options

Options Hash (options):

  • (String) :category N/A —

    The id of a job category to limit to. See aj.categories.getList

  • (String) :type N/A —

    The id of a job type to limit to. See aj.types.getList

  • (String) :sort — default: 'date-posted-desc' —

    Accepted values are 'date-posted-desc' and 'date-posted-asc'

  • (String) :company N/A —

    Free-text matching against company names. Suggested values are the ids from aj.jobs.getCompanies

  • (String) :location N/A —

    Free-text matching against company location names. Suggested values are the ids from aj.jobs.getLocation

  • (String) :keywords N/A —

    Keywords to look for in the title or description of the job posting. Separate multiple keywords with commas. Multiple keywords will be treated as an OR

  • (Integer) :page — default: 1 —

    The page of listings to return. Defaults to 1.

  • (Integer) :perpage — default: 1 —

    The number of listings per page. The default value is 10. The maximum value is 100.
